# bpe.geomatik

## Problématique
D'un côté nous disposons de fichiers de l'INSEE indiquant le nombre d'équipement à la commune, un chifre différent de zéro indiquant le nombre d'é"quipement pour la commune. Ces fichiers appelés BPE sont mis en ligne annuellement.
De l'autre côté des fichiers plus ou moins récents (2009 pour certains, 2014 pour d'autres) indiquant le temps d'accès pour une commune à un "équipement" bien particulier (ex: maternité, généraliste).

Mais aucun fichier regroupant la richesse des équipements recensés par l'INSEE et la ditance ou le temps de parcours jusqu'à l'équipement pour une commune ne disposant pas de l'équipement en question.
